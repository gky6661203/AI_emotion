# AI Emotion Campus Tree Hole

## One-click backend start

In PowerShell, run:

```powershell
.\start-backend.ps1
```

This script will:
- Enter `services/ts-api`
- Install dependencies if `node_modules` is missing
- Start the backend in development mode

## AI companion model config

If you want the chat replies to be generated by an external model, set these environment variables in `services/ts-api/.env`:

```env
BAILIAN_API_KEY=your_api_key_here
BAILIAN_API_URL=https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions
BAILIAN_MODEL=qwen-plus
```

If `BAILIAN_API_KEY` is missing, the backend will fall back to the built-in supportive reply.
